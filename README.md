# SIC_GATYA

2023SIC_ハッカソン制作用レポジトリ

# Requirement

使用ライブラリ
* Python 3.10.9
* click==8.1.3
* colorama==0.4.6
* Flask==2.2.2
* Flask-Login==0.6.2
* Flask-SQLAlchemy==3.0.3
* geenlet==2.0.2
* itsdangerous==2.1.2
* Jinja2==3.1.2
* MarkupSafe==2.1.2
* SQLAlchemy==2.0.3
* typing_extensions==4.5.0
* Werkzeug==2.2.2
* wincertstore==0.2

# Installation

使用ライブラリのインストール
```bash
pip install requirements.txt
```
# Usage

サーバーの起動方法
```bash
git clone https://github.com/sintaro-katuta/SIC_GATYA.git
cd SIC_GATYA
flask run
```
